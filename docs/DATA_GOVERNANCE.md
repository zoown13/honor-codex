# 데이터 수집·검수 정책

## 출처 우선순위

1. 병무청 예우시설 JSONP와 개별 상세
2. 병무청 병역명문가 공지
3. 법제처 자치법규 OPEN API
4. 운영자가 공식 원문을 확인한 수동 입력

각 혜택은 원천 식별자, 원문 URL, 수집시각, content hash, 근거 조항 또는 원문 발췌를 보존합니다.

## 자동 게시

다음 변경만 모든 품질 검사를 통과하면 자동 승인합니다.

- 공식 원천에 새로 등장한 시설
- 전화번호와 홈페이지
- 분류명과 표시 주소의 비핵심 정정
- 동일 행정구역 안에서의 유효한 좌표 정정

## 필수 검수

다음 변경은 소유자가 원문을 확인하고 승인해야 합니다.

- 시설 삭제·종료 또는 행정구역 이동
- 할인율·면제 범위·대상자·증빙·이용 절차·유효기간
- 전국 혜택과 조례의 신규 요약 또는 수정
- 전체 건수 5% 이상 변화 또는 삭제 1% 초과
- 원천 간 내용 충돌, 필수 필드 누락, 잘못된 좌표

원문에 없는 내용은 보충하지 않고 `공식 문서에 명시되지 않음—시설 또는 담당부서 확인 필요`로 표시합니다.

## 보존과 게시

- 원문 스냅샷: 180일
- Lambda/API 로그: 14일
- 활성·직전 dataset: 항상 보존
- 승인된 변경만 알림 이벤트 생성
- 게시 실패 시 manifest를 바꾸지 않고 직전 dataset 유지

## 사용 조건

병무청 JSONP는 공개 화면에서 호출되지만 정식 버전과 SLA가 명시된 API가 아닙니다. 파일럿은 저빈도 캐시와 출처 표시를 전제로 하며, 공개 확대 전 자동수집과 재사용 조건을 제공기관에 확인해야 합니다.
